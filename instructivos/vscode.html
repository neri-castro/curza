¿Qué es Visual Studio Code?

Visual Studio Code (o VS Code) es un editor de código fuente desarrollado por Microsoft. No hay que confundirlo con su "hermano mayor", Visual Studio, que es un Entorno de Desarrollo Integrado (IDE) completo. VS Code se centra en ser un editor de texto súper vitaminado, rápido y altamente personalizable.

Lo que realmente lo hace especial es su increíble flexibilidad. Gracias a un vasto ecosistema de extensiones, puedes adaptarlo para trabajar con casi cualquier lenguaje de programación, desde desarrollo web (HTML, CSS, JavaScript) hasta Python, Java, C++, y muchos más.

Instalación Paso a Paso

Elige tu sistema operativo. Te guiaremos en el proceso para que tengas VS Code funcionando en minutos.

La forma más sencilla de instalar VS Code en Windows es usando el instalador. Sigue estos pasos:

  1. Descargar el instalador: Ve al sitio web oficial y descarga el instalador de usuario para Windows.
  2. Ejecutar el instalador: Abre el archivo VSCodeUserSetup-x64.exe que acabas de descargar.
  3. Acuerdo de licencia: Lee y acepta el acuerdo de licencia para continuar con la instalación.
  4. Opciones de instalación: Se recomienda dejar las opciones por defecto, que incluyen agregar VS Code al menú contextual del explorador y al PATH del sistema, lo que facilita su uso desde la terminal.
  5. Finalizar la instalación: Haz clic en "Instalar" y, una vez completado, haz clic en "Finalizar". VS Code se iniciará automáticamente.

Para sistemas basados en Debian (como Ubuntu), la forma más sencilla es usar el repositorio de paquetes de Microsoft. Esto te asegura que siempre tendrás la última versión y actualizaciones automáticas. Abre una terminal y ejecuta estos comandos:

  1. Descargar e instalar la clave GPG de Microsoft: 
    sudo apt-get install wget gpg
wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
sudo install -D -o root -g root -m 644 packages.microsoft.gpg /etc/apt/keyrings/packages.microsoft.gpg
rm packages.microsoft.gpg
  2. Agregar el repositorio de VS Code: 
    sudo sh -c 'echo "deb [arch=amd64,arm64,armhf signed-by=/etc/apt/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
  3. Actualizar el índice de paquetes e instalar: 
    sudo apt-get update
sudo apt-get install code
  4. Iniciar VS Code: Puedes iniciar el editor desde el menú de aplicaciones o escribiendo code en la terminal.

Primeros Pasos y Extensiones

Ya tienes VS Code. Ahora, vamos a potenciarlo con algunas de las extensiones más populares y útiles.

Atajos y Funciones Clave

  • Paleta de Comandos (Ctrl+Shift+P): Tu centro de control. Busca y ejecuta cualquier comando sin necesidad de usar el ratón.
  • Terminal Integrada (Ctrl+`): Abre una terminal directamente en VS Code. Súper útil para ejecutar comandos de Git, npm, etc.
  • Buscador de Archivos (Ctrl+P): Encuentra y abre cualquier archivo en tu proyecto de forma instantánea.

Extensiones Esenciales

Spanish Language Pack

Traduce la interfaz de VS Code al español para una experiencia más cómoda.

Live Server

Lanza un servidor de desarrollo local con recarga en vivo para páginas estáticas y dinámicas.

GitLens

Potencia las capacidades de Git integradas, mostrando quién modificó cada línea de código y cuándo.
